Los geht's!

Als Erstes benötigen Sie einen persönlichen, geheimen Schlüssel. Erstellen Sie einen im Optionen-Menü unter "Schlüssel" oder importieren Sie existiernde geheime Schlüssel. Danach können Sie öffentliche Schlüssel von Freunden oder Bekannten herunterladen oder sie via QR Codes oder NFC austauschen.

Es empfiehlt sich, dass Sie OI File Manager für komfortable Dateiauswahl sowie Barcode Scanner um QR Codes scannen zu können. Wenn Sie auf die Links klicken, öffnet sich entweder der Google Play Store oder F-Droid für die weitere Installation, abhängig davon was auf Ihrem Gerät installiert ist..
